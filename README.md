# NerfOrBuff.ai
Analyseur de Patch Notes Gaming boosté à l'IA
### Description
**NerfOrBuff.ai** est une application web conçue pour les joueurs qui souhaitent comprendre instantanément l'impact des mises à jour sur leur jeux  préfèrés. Grâce à l'IA, l'outil analyse les aptchs notes complexes pour en extraire l'essentiel
> **Projet réalisé dans le cadre de ma recherche d'atlernance dans le developpement web.**

---

### Fonctionnalités IA
* **Analyse de Sentiment :** Détermine si un changement est un "Buff" ou un "Nerf".
* **Résumé contextuel :** Génère un "TL;DR" spécifique à chaque personnage ou arme.
* **Prédiction de META :** Analyse l'impact global sur l'équilibre du jeu via LLM.

### Stack Technique (En cours de développement)
* **Frontend :** React.js + Tailwind CSS
* **Backend :** Python (FastAPI)
* **IA :** API OpenAI / Gemini
* **Scrapping :** BeautifulSoup4
